# 送迎レーン デモ

送迎計画、ドライバー操作、進捗、実績確認を行うローカルデモです。

現在の方針では、アプリ内データを正とします。Excelはエクスポート形式として扱い、日々の管理は画面上のマスタ管理・計画作成で行います。

## 起動

```powershell
npm install
npm run db:push
npm run dev
```

ブラウザで `http://localhost:3000` を開きます。

## デモ手順

0. `現場シミュレーション` で現場データを投入し、検証手順を確認する
1. `マスタ` で利用者、車両、職員を登録する
2. `計画作成` で送迎便を作成し、利用者を停車順に追加する
3. `配車計画` で送迎便を選択し、予定時刻、備考、停車順、キャンセル、削除を確認する
4. `ドライバー` で到着、乗車、出発、完了を順に記録する
5. `ダッシュボード` と `進捗` で運行状況を確認する
6. `実績` で予定と実績の差分を確認する
7. `Excel出力` で送迎表形式の先頭シートを含む `.xlsx` を出力する

登録データは Prisma 経由で SQLite の `prisma/dev.db` に保存されます。再読み込みしてもDBから復元されます。

## 現場シミュレーション

`現場シミュレーション` では、雨天、車いす、薬確認、当日キャンセル、複数車両、AM/PM便を含む一日分のデータをDBへ投入できます。
投入後は、配車計画、ドライバー操作、進捗、実績、Excel出力を順番に確認します。
デモ前に状態を戻したい場合は、同画面の `デモDBを空にする` を使います。

## 注意

- 実 Excel ファイルはリポジトリに入れません。
- デモ表示では氏名、電話、住所をマスクできます。
- 初回デモは地図、GPS、通知、AI最適化を含みません。
