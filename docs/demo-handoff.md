# デモ送付メモ

## 送付文面

以下の文面をそのまま送れます。

```
送迎支援アプリのMVPデモを用意しました。

確認ポイント:
- 「今日の運行」から、朝の確認、車での操作、施設側の進捗確認、実績出力まで順番に確認できます
- 雨天・車いす・薬確認・キャンセルを含む一日分の現場データを最初から確認できます
- 利用者・車両・職員マスタを登録/編集/削除できます
- AM/PMや車両別の送迎便を選択できます
- 停車順、予定時刻、備考、キャンセル、削除を編集できます
- ドライバー画面で「到着した」「乗った/降りた」「出発する」「完了した」を大きなボタンで記録できます
- 進捗、実績、Excel送迎表出力まで確認できます

デモURL:
https://refu15.github.io/send-demo/

おすすめ確認手順:
1. ログイン画面で「管理者」を選び、パスワード入力後に「今日の運行」を確認する
2. 「配車を確認」から、車両、担当者、停車順、予定時刻を見る
3. 画面上部のロール切替で「ドライバー」を選ぶ
4. 「ドライバー画面」で1名分を「到着した」から「完了した」まで進める
5. 「施設職員」を選び、「進捗」で残り人数、送迎中、完了、キャンセルを見る
6. 「実績を見る」で、予定時刻と実績時刻を確認する
7. 「出力する」または「Excel出力」で、送迎表とCSV出力を確認する
```

## GitHub Pagesで送る場合

GitHub Pages用の静的デモ版は `docs/deploy-github-pages.md` の手順で公開します。
公開後、Actionsに表示されるURLを上の `デモURL` に貼って送付します。

## デモ前チェック

- `今日の運行` が最初に表示される
- ログイン画面で `管理者`、`施設職員`、`ドライバー` を選べる
- `管理者` は `今日の運行`、`施設職員` は `進捗`、`ドライバー` は `ドライバー画面` に入れる
- `今日の運行` から `配車を確認`、`ドライバー画面`、`進捗を見る`、`実績を見る`、`出力する` へ進める
- `マスタ` で利用者・車両・職員の編集/削除が見える
- `配車計画` で送迎便選択、停車順上下、時刻/備考保存、キャンセル、削除ができる
- `ドライバー` で到着、乗車/降車、出発、完了、直前取消、連絡先確認ができる
- スマホ幅でも主要ボタンの文字が切れず、押しやすい大きさで表示される
- `Excel出力` で先頭シート `送迎表` が出る
