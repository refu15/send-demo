# デモ送付メモ

## 送付文面

以下の文面をそのまま送れます。

```
送迎支援アプリのMVPデモを用意しました。

確認ポイント:
- 現場シミュレーションから、雨天・車いす・薬確認・キャンセルを含む一日分のデータを投入できます
- 利用者・車両・職員マスタを登録/編集/削除できます
- AM/PMや車両別の送迎便を選択できます
- 停車順、予定時刻、備考、キャンセル、削除を編集できます
- ドライバー画面で到着、乗車、出発、完了を記録できます
- 進捗、実績、Excel送迎表出力まで確認できます

デモURL:
（ここにURLを貼ってください）

おすすめ確認手順:
1. 現場シミュレーションを開く
2. 現場データを投入する
3. 配車計画で送迎便を切り替え、停車順や時刻を修正する
4. ドライバー画面で1名分を完了まで進める
5. 進捗、実績、Excel出力を確認する
```

## ローカル起動

```powershell
cd C:\Users\nnkre\send-demo
npm install
npm run db:push
npm run dev
```

ブラウザで `http://localhost:3000` を開きます。

## 本番に近い確認

```powershell
cd C:\Users\nnkre\send-demo
npm run demo:prepare
npm run start
```

## GitHub Pagesで送る場合

GitHub Pages用の静的デモ版は `docs/deploy-github-pages.md` の手順で公開します。
公開後、Actionsに表示されるURLを上の `デモURL` に貼って送付します。

## デモ前チェック

- `現場シミュレーション` で `現場データを投入`
- `マスタ` で利用者・車両・職員の編集/削除が見える
- `配車計画` で送迎便選択、停車順上下、時刻/備考保存、キャンセル、削除ができる
- `ドライバー` で到着、乗車、出発、完了、取り消しができる
- `Excel出力` で先頭シート `送迎表` が出る
